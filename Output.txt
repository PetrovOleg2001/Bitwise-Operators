Bitwise Operators in Java
-------------------------
a = 173; b = 212
Bitwise OR (|):
a | b = 173 | 212 = 10101101 | 11010100 = 11111101 = 253
Bitwise AND (&):
a & b = 173 & 212 = 10101101 & 11010100 = 10000100 = 132
Bitwise XOR (^):
a ^ b = 173 ^ 212 = 10101101 ^ 11010100 = 1111001 = 121
Bitwise Complement (~):
~a = 173 =  ~ 10101101 = 11111111111111111111111101010010 = -174
Signed Right shift operator (>>):
a >> 2 = 173 >> 2 = 10101101 >> 2 = 101011 = 43
Signed Right shift operator (<<):
a << 2 = 173 << 2 = 10101101 << 2 = 1010110100 = 692
